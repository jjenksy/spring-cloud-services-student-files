Personal Experience

Experience with:

  1. Spring, Spring Boot even better
  2. Git

Pivotal Cloud Foundry Environment

  1. Access to a Pivotal Cloud Foundry Environment (an environment is not provided as part of the labs)
  2. Spring Cloud Services 1.1.1 Tile installed in the Pivotal Cloud Foundry Environment
  3. 6 GB of Memory Quota available

Local machine requirements

  1. 4GB Memory
  2. Java: JDK 1.8
  3. Maven
  4. git
  5. A GitHub Account
  6. curl
  7. The CloudFoundry CLI: cf

If you are on a Mac and using homebrew, you can install the cloudfoundry cli as follows:

$ brew tap cloudfoundry/tap
$ brew install cf-cli

Optional (but recommended)

  1. SpringSource Tool Suite
  2. Json Formatter
